# Induced Dependence

## A Unified Operational Framework across AI, Bell-Type Filtering, Biomedical Selection Systems, and Ecological Observation

Induced Dependence is an operational framework for studying how filtering, selection, and observational constraints deform probabilistic states across scientific systems.

The framework introduces a geometric interpretation of filtering processes through the operator:

\[
\mathcal{F}:P(R)\rightarrow P(R|F=1)
\]

and diagnoses when observational filters become informative about response variables through the condition:

\[
I(R;F)>0
\]

This repository contains:

- the manuscript,
- reproducible simulations,
- regression analyses,
- robustness diagnostics,
- ecological observational frameworks,
- and probabilistic filtering geometry experiments.
